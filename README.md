# Age & Gender Detection with Data Analysis

This project is a web application built using Python, OpenCV, and Streamlit.

## Features
- Detects age and gender from image upload or camera
- Displays result on image
- Collects prediction data
- Shows data analysis using charts

## Technologies Used
- Python
- OpenCV
- Streamlit
- Pandas

## How to Run
1. Install required libraries
2. Run the command:
   streamlit run app.py

## Output
- Age and Gender prediction
- Data table
- Graphs for analysis
